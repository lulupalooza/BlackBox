void taskPID ( )

void taskPID()

This task compares the temperature value in temp_queue, or the measured temperature inside the black box, and compares it to the desired black box temperature. Using the difference between measured and desired temp, a new ratio of heater on/off time is created. This ratio is stored in PWM_ratio for use by the power driver module to control the heater and adjust black box temp as needed. The desired temp will default to DEFAULT_TEMP unless the user has specified a desired temp.

Returns
None.
